🔄 Actualizar con nuevas medidas
1. Exporta el Excel de Measure Killer con todos los objetos del modelo.
2. Pide a una IA: "Genera arrays NODES y EDGES para D3.js desde este Excel, filtrando Type=Measure y Used_in_object_type=Measure."
3. Reemplaza const NODES y const EDGES en este archivo HTML.
4. Haz click en ✏️ para actualizar el título si es necesario.
